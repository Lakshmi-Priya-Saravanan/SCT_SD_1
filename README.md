A JavaFX-based GUI application that converts temperatures between Celsius, Fahrenheit, and Kelvin.

## Technologies Used
- Java 21
- JavaFX 21
- IntelliJ IDEA

## How to Run
1. Open project in IntelliJ
2. Set VM options:
   --module-path "C:\Program Files\Java\javafx-sdk-21.0.10\lib" --add-modules javafx.controls
3. Run MainApp
